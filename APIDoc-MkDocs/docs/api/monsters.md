# Monster Properties

* name : String
* maxHealth : Integer
* moveRange : Integer

# Monster Queries

### Get All Monsters

    GET /api/monsters

### Get Specific Monster

    GET /api/monsters/{id}

### Create Monster

    POST /api/monsters/

### Update Monster

    PUT /api/monsters/{id}

### Delete Monster

    DEL /api/monsters/{id}

# Aggregate Monster Information

Currently there is no nested information within monsters
